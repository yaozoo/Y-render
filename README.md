## 安装依赖

```
npm install
```

or

```
yarn
```

### 本地启动

```
npm run dev
```

### 打包

```
npm run build
```

### 发布 npm

```
npm run build_lib
```

### 使用教程

```
下载依赖
npm i y-render -D

main.ts引入并注入
import YRender from "y-render";
app.use(YRender);
```
